Below are some pages that this app include ####
 
1. App Walkthrough Slider Page. 
2. Login & Signup Page. 
3. Dashboard Page With Slider. 
4. Video Consultaion & Booking Page.  
5. Text Consultation & Booking Page. 
6. Live Chat Page. 
7. Profile Page. 
8. Blog Post with Share Actions Page. 
9. Notification Page.
